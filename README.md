# C01N
C01N project

To get a basic flashing station make sure you have these dependencies

-> python3

-> pip3

-> adafruit-ampy (through pip3)

-> esptool (through pip3)

Then get our stuff

$ git clone https://github.com/C01N-GovTech/C01N

$ cd uPy_C01N_OS

$ chmod +x flash.sh

$ ./flash.sh /dev/"yourC01Nlocation"
